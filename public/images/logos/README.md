# Logo Files

This directory contains the G&L Quality Goods brand logos.

## Logo Files

1. **logo-1.png** - Simple serif font logo
   - G&L in elegant serif font
   - "QUALITY GOODS" in sans-serif below
   - Black background, white text, gold ampersand

2. **logo-2.png** - **CURRENTLY IN USE** - Handwritten style with palm trees
   - G&L in brush-stroke/handwritten style
   - Two stylized palm trees in gold (trunks crossing in X shape)
   - "QUALITY GOODS" curved along an arc below
   - Black background, white text, gold accents

3. **logo-3.png** - Handwritten style (simple)
   - G&L in stylized brush-stroke script
   - "QUALITY GOODS" in curved arc below
   - Black background, white text, gold ampersand

## Usage

The second logo (logo-2.png) is currently being used on the website. To switch logos, update the image reference in:
- `components/Navbar.tsx` (Next.js version)
- `index.html` (static HTML version)
